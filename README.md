# Hotel Reservation Platform

Full-stack aplikacija za rezervaciju hotelskih soba.

## Stack

Backend:
- Node.js
- Express
- TypeScript
- TypeORM
- MySQL
- JWT
- bcrypt

Frontend:
- Angular
- Angular Material
- TypeScript
- Reactive Forms
- Angular Router

## Funkcionalnosti

- Registracija i prijava korisnika
- JWT autentifikacija
- Pretraga soba po datumu, broju gostiju, tipu sobe i pogodnostima
- Kreiranje rezervacije
- Pregled rezervacija
- Otkazivanje rezervacije
- Admin dashboard
- Upravljanje sobama
- Izmena korisničkog profila

## Pokretanje

Backend: 

```bash

npm run dev
npm run build

Frontend:

npm install
npm start
